CSS Mask-Image Explanation

What is mask-image?

mask-image is a CSS property that uses an image to define which parts of an element are visible.

How it works:
  • White areas in the mask = fully visible (shows the element)
  • Black areas in the mask = fully transparent (hides the element, shows what's behind)
  • Gray areas = partially visible (semi-transparent)

mask-image vs -webkit-mask-image

-webkit-mask-image is the vendor-prefixed version for older browsers (Safari, Chrome before full support). Both should be included for maximum browser compatibility. Modern browsers support mask-image without the prefix.

Data URI Format

The SVG is embedded directly in the CSS as a data URI:

data:image/svg+xml,%3Csvg...

This means:

  • data: - indicates a data URI
  • image/svg+xml - the MIME type (SVG image)
  • %3Csvg... - the SVG code, URL-encoded

To decode and view: Copy the data URI from the CSS, paste it in your browser's address bar, or use an online data URI decoder.

CSS Properties Used

  • mask-image / -webkit-mask-image - the SVG mask
  • mask-repeat: repeat-x - repeats the pattern horizontally
  • mask-size: 130px 75px - sets the size of each pattern tile
  • background-color: var(--color-primary) - the color that shows through white areas
